💳 Caixa Eletrônico em Python

Projeto desenvolvido para praticar estruturas de repetição, condicionais e manipulação de variáveis em Python.

📌 Sobre o projeto

Este programa simula um caixa eletrônico simples no terminal.
O usuário começa com um saldo inicial de R$1000,00 e pode:

✔️ Consultar saldo

✔️ Realizar saques (com validação de saldo)

✔️ Realizar depósitos

✔️ Encerrar o programa

O sistema utiliza:

Loop while para manter o menu ativo

Estruturas condicionais (if/elif)

Manipulação de variáveis numéricas

Validação básica de entrada do usuário
